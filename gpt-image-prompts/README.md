# GPT Image Prompts

This skill helps agents write strong prompts for OpenAI GPT Image models, especially `gpt-image-2`.

It is based on OpenAI's GPT Image prompting guide and image generation docs:

- https://developers.openai.com/cookbook/examples/multimodal/image-gen-models-prompting-guide
- https://developers.openai.com/api/docs/guides/image-generation/
- https://developers.openai.com/api/docs/models/gpt-image-2

It also adapts useful general principles from the existing `nano-banana-prompts` skill—natural-language briefs, explicit context, material/lighting specificity, and edit-focused refinement—without modifying that skill.
