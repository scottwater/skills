# Test Behavior Reviewer

## Search

Changed behavior that existing tests cannot distinguish; missing failure, boundary, contract, rollback, or concurrency scenarios; mocks that bypass the relevant path; assertions that permit false positives; and nondeterministic test design.

## Evidence bar

Changed code path, concrete failure scenario, relevant existing test or precise gap, and the observable assertion that would distinguish correct from incorrect behavior.

## Exclude

Coverage percentages, tests for every line, test style preferences, and requests for tests without a concrete behavioral risk.
